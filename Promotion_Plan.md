## 2. 推广渠道与策略

### 2.1 编程猫官方社区 (渗透式推广)
*   **策略**: 不要直接发硬广（容易被封号）。采用“技术分享”或“作品展示”的形式。
*   **内容示例**:
    *   发布一个帖子：“我用 Python/Vue 写了一个第三方社区客户端，大家来看看效果”，视频演示中露出网址。
    *   在自己的个人简介（签名档）中隐晦加入链接（如：`Base: bettermao.cfd`）。
    *   在热门技术贴下回复：“这个源码可以在 CodeMan 上找到备份”。

### 2.2 QQ 群 / 微信群 (病毒式传播)
这是最直接有效的渠道，因为编程猫用户在 QQ 群极其活跃。
*   **目标群**: 编程猫官方群、作品自荐群、素材交易群、黑客/技术交流群。
*   **话术**:
    *   “兄弟们，发现一个新站 `bettermao.cfd`，可以直接下源码，不用求人了。”
    *   “官方社区又崩了？来这边聊 -> `bettermao.cfd`”
    *   “新开了个 API 接口，想做第三方应用的进来看。”
*   **动作**: 制作一张长图海报，包含网站截图、功能列表和二维码，在群里转发。

### 2.3 Bilibili 视频推广 (视觉冲击)

### 2.4 邀请制与 KOL 合作

## 4. 风险控制 (注意事项)

1.  **合规性**: 确保网站底部的“免责声明”清晰，强调非官方身份，避免商标侵权纠纷（虽然用了 BetterMao，但在 About 页已注明是非官方）。
2.  **防爆破**: 刚推广时可能会引来“小学生黑客”进行 DDoS 或 XSS 测试。
